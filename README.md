# 💈 B.B HairStyle — PWA

אפליקציית קביעת תורים שניתן להתקין על כל טלפון (אנדרואיד + אייפון).

---

## 📁 קבצים בתיקייה
- `index.html` — האפליקציה המלאה
- `manifest.json` — הגדרות PWA
- `sw.js` — Service Worker (מאפשר עבודה offline)
- `icon.svg` — אייקון האפליקציה

---

## 🚀 העלאה לאינטרנט (חינם — 5 דקות)

### אפשרות 1: Netlify (הכי קל)
1. כנס לאתר: https://netlify.com
2. לחץ "Sign up" (חינם)
3. גרור את **כל התיקייה** לאזור ה-Drag & Drop
4. מקבל קישור כמו: `https://bb-hairstyle.netlify.app`
5. שלח את הקישור ללקוחות!

### אפשרות 2: GitHub Pages (חינם)
1. https://github.com → צור חשבון
2. New Repository → שם: `bb-hairstyle`
3. העלה את הקבצים
4. Settings → Pages → Source: main
5. האתר יהיה ב: `https://USERNAME.github.io/bb-hairstyle`

---

## 📲 איך לקוחות מתקינים

### אנדרואיד (Chrome):
כשנכנסים לאתר → מופיע באנר "התקן עכשיו" → לוחצים → האפליקציה מותקנת!

### אייפון (Safari):
1. פתח את הקישור ב-Safari
2. לחץ כפתור השיתוף ⬆️ 
3. בחר "הוסף למסך הבית"
4. לחץ "הוסף"
